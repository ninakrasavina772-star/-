# Редактор фото

Вставьте ссылку на картинку → **Улучшение** → скачивание JPG (нейросеть Real-ESRGAN или быстрый режим).

## Локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

Откройте в браузере: `http://localhost:8501`

## Публичная ссылка (Streamlit Cloud)

1. Создайте репозиторий на [GitHub](https://github.com/new) и загрузите **содержимое** этой папки (не обязательно переименовывать репозиторий — важно только имя папки у себя на диске).
2. Зайдите на [share.streamlit.io](https://share.streamlit.io), войдите через GitHub.
3. **New app** → выберите репозиторий → **Main file path:** `app.py` → Deploy.
4. Через минуту появится ссылка вида `https://ваш-проект.streamlit.app`.

Первый запуск в облаке скачает модель (~33 МБ); нейросеть на CPU может работать медленно — при нехватке памяти в боковой панели выберите режим **Быстро**.
